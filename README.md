# Material para Curso Data Science

El repositorio ha sido desarrollado a partir de mi experiencia dictando talleres y seminarios sobre analítica avanzada. Por esta razón cuenta con recursos para el desarrollo de habilidades técnicas necesarias en la carrera de ciencia de datos. Permite comprender y practicar desde conceptos básicos de programación, hasta aspectos de aprendizaje computacional avanzado, abarcando las siguientes temáticas a partir de ejercicios y casos de uso aplicados:
* Fundamentos de Python y análisis de datos (6 capítulos).
* Herramientas complementarias para ciencia de datos (2 capítulos).
* Fundamentos del aprendizaje computacional (5 capítulos).
* Métodos matemáticos para aprendizaje computacional (3 capítulos).
* Temas avanzados de aprendizaje computacional (3 capítulos). 

## Contenido del repositorio

* En el directorio **notebooks** se encuentran Jupyter Notebooks (*.pynb*) organizados por secciones y capítulos (Ver *README* del directorio). En cada uno de estos archivos es posible encontrar casos aplicados a situaciones reales, o ejercicios enfocados en el desarrollo de capacidades para programación y ciencia de datos.   
    
* En el directorio **datasets** se encuentran todos los ficheros (*.csv*, *.xlsx*, *.txt*) a utilizar en el desarrollo de los capitulos y sus correspondientes notebooks. Esta información ha sido debidamente procesada para que su uso sea público y por tanto pueda emplearse libremente más allá de lo descrito en las temáticas aquí tratadas. 

* En el directorio **complementary_files** se encuentran archivos diversos que sirven de insumo para las explicaciones y demás contenido de los notebooks. Es por tanto una carpeta de material auxiliar que por si solo no aporta ningún valor agregado.

## Lenguajes utilizados en recursos

* Python (Ver *py_reqs* para consultar sobre módulos y librerías utilizadas en los notebooks)
* SQL (En específico PostgreSQL)
* Powershell / Bash

## Herramientas sugeridas

* [Python](https://www.python.org/downloads/) versión 3.13 o superior.
* [Visual Studio Code](https://code.visualstudio.com/Download?_exp_download=fb315fc982) como editor de código fuente. Aquí conviene instalar las extensiones de Python, Jupyter y PostgreSQL.
* [PostgreSQL](postgresql.org/download/) como administrador de bases de datos.
