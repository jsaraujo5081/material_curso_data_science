# Contenido del directorio

## Sprint 1 - Python Básico I (Tiempo estimado: 5-6 horas)

* **S01_Python_Basico_I_Sesiones**: Ejercicios a realizar durante sesiones con el tutor sobre manejo de variables, gestión de errores, manipulación de cadenas de texto y manejo de listas.
* **S01_Python_Basico_I_Ejercicios**: Ejercicios extra sobre estos mismos temas para ser resueltos sin apoyo de un tutor, permitiendo al estudiante alcanzar mayor seguridad sobre la sintaxis del lenguaje de programación.

## Sprint 2 - Python Básico II (Tiempo estimado: 5-6 horas)

* **S02_Python_Basico_II_Sesiones**: Ejercicios a realizar duraante sesiones con el tutor sobre manejo de diccionarios, creación de funciones, uso de bucles y uso de condicionales.
* **S02_Python_Basico_II_Ejercicios**: Ejercicios extra sobre estos mismos temas para ser resueltos sin apoyo de un tutor, permitiendo al estudiante alcanzar mayor seguridad sobre la sintaxis del lenguaje de programación.
